# Guía de Publicación - Ontología de Ciberseguridad

## Descripción
Este repositorio contiene una ontología de ciberseguridad desarrollada en OWL/RDF,
un agente inteligente en Python y consultas SPARQL de ejemplo.

## Estructura del repositorio
- `ontologia.owl` — Ontología principal en formato OWL/RDF
- `agente_python.py` — Agente inteligente que razona sobre la ontología
- `ejemplos/consultas_sparql.md` — Consultas SPARQL de ejemplo
- `docs/guia_publicacion.md` — Este archivo

## Requisitos
- Python 3.x
- rdflib (`pip install rdflib`)
